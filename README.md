# TextToImg
基于Qt的文字转图片工具（text to image）

工具栏中所有的按钮都有生成图片的功能，图片命名方式为“文字+时间戳.png”，其中，若文字有多行，则取第一行文字。
所有图片会生成到根目录下的images文件夹中。
